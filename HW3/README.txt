In this directory are two c++ source files, HW3.cpp and HW3_ExtraCredit.cpp.

HW3.cpp contains the implementation for question 1's special bubble sort and
can be run with the command "./HW3".

HW3_ExtraCredit.cpp contains the implementation for the extra credit question,
which is a partition algorithm meant for arrays with multiple duplicate values.
This can be run with the command "./EC".
